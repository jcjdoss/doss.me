About This Site

How it's built and why I'm transparent about AI.

AI Collaboration

This site was built with AI assistance. The writing was refined by AI. The code was generated through conversation with AI tools.

I'm explicit about this because hiding it seems pointless. AI is a tool. I use tools that help me work more effectively.

  • Ideas and direction — mine
  • Writing and organization — AI-assisted from my input
  • Code and implementation — AI-generated from my requirements
  • Final review — me, checking that it actually represents what I mean

Why Transparency

Most people obscure their AI usage. I don't see the point.

Using AI well is a skill. The people I'd want to work with understand that. The people who would judge me for being honest about my tools are probably not people I need to impress.

I'm more interested in outcomes than in performing effort.

Process

1

Input

I explain what I'm thinking in conversation—voice notes, chat, whatever's convenient.

2

Organization

AI structures my thoughts into something coherent. Better grammar, clearer structure.

3

Review

I read it back. Does it sound like me? Is it accurate? Would I actually say this?

4

Publish

Good enough ships. Perfect never does.

Technical Details

  • AI Model — Claude (Anthropic)
  • Frontend — Plain HTML and CSS
  • Hosting — GitHub Pages
